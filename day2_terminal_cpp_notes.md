# Day 2 Notes — Terminal and First C++ Program

## What I learned

Today I learned how to use basic terminal commands, create files, move between folders, and compile/run my first C++ program.

## Commands I used

### ls
Shows files and folders in the current directory.

### cd folder_name
Moves into a folder.

### cd ..
Moves one folder back.

### New-Item filename -ItemType File
Creates a new file in PowerShell.

### g++ hello.cpp -o hello
Compiles a C++ file into an executable program.

### .\hello.exe
Runs the compiled C++ program on Windows PowerShell.

## C++ concepts I learned

- `#include <iostream>` allows input/output.
- `int main()` is where the program starts.
- `cout` prints text.
- `endl` moves to a new line.
- `return 0;` means the program ended successfully.

## Mistake I fixed

My `hello.cpp` file was in the wrong folder, so the compiler could not find it.

## Lesson

The terminal only works based on the folder I am currently in. File location matters.
